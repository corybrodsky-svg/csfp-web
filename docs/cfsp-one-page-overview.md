# Conflict-Free SP (CFSP)

## One-line positioning

A simulation operations command center for SP staffing, readiness, schedules, materials, and event-day execution.

## The problem

Simulation teams often run high-stakes events through spreadsheets, email threads, memory, and last-minute fixes. Staffing gaps, room changes, late materials, unclear release timing, and day-of questions can turn routine events into fragile operations.

## The solution

CFSP centralizes event setup, SP staffing, confirmations, release gates, training materials, schedule previews, readiness, and SP portal access. The goal is to give simulation teams one shared operational picture before, during, and after an event.

## Built for

SP Program Directors, Simulation Operations Specialists, Clinical Skills teams, Nursing/PA/Medical Education programs, and simulation center leadership.

## Core capabilities

- SP staffing and confirmation tracking
- Schedule and room flow visibility
- Training/materials release controls
- Readiness checklist and blocked-state visibility
- SP confirmed-work portal
- Event-day check-in/readiness support

## Demo scenario

Nursing IPE Simulation - 10 SPs, 4 rooms, 48 learners, role/case assignments, training materials, schedule preview, and release readiness.

## Pilot-safe note

CFSP prototype/pilot use should rely on fictional or approved demo data only. It is not intended for PHI, real patient records, or unauthorized confidential institutional/student data.
